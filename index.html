HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element.
  • The title defines the title for the webpage which is very important for SEO.
  • Lang = "en" tells the browser the language for the webpage. "En" stands for English.
  • Child elements are ones that are nested inside other elements.
  • Parest elements are ones that hold all the children.
  • Semantic HTML is the name of elements that describe its content and role it plays on a webpage.
  • Not all elements are semantic.
  • It's best practice to use semantic elements wherever possible.
  • We insert the contect between an opening tag and a closing tag so the browser understands how to treat content.
  • Closing tags begin with a /.
  • An element refers to an entire element.
  • A tag refers to only what appears inside the angle brackets.
  • The words 'element' and 'tag' are commonly used interchangeably.
  • Alt text is important for screenreaders and other assistive technologies. It can improve search engine rankings.
  • Developers text code early to often catch bugs. Sometimes this is as siple as looking for changes in the web browser.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * selector is a wild card symbol that declares all the elements of the page will have the style properties set at "".
  • You can combine 2 selectors into a single CSS rule with a comma allowing the declarations to affect both selectors.
  • You need to link the CSS to the HMTL file.
  • Add style by using class attributes.
  • Class attributes can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned to the CSS rule.
  • Card - designates this a class selector. Any HTML element assigned to the class card will gain these property assignments.
  • Assigning colours in CSS can be done in many ways including semantic references such as "grey" or we can also refer to a colour by its hexadecimal colour.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git must precede every git statement
  • git checkout -b feature/"": creates a new feature branch
  • git add -A: indicates we want to make changes
  • git commit -m"": describes what the commit contains
  • Do not code in the main or master branch!
  • Pull requests are required to make changes to the main branch
  • Always remember to stage files after saving any changes

JavaScript

  • JavaScript is a high level dynamic programming language that allows users to interact with websites.
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • A variable is a named container that allows us to store data in our code.
  • A variable can contain a string. Strings are used for storing and manipulating text and are surrounded by quotation marks in code.
  • To access a string you can simply refer to the variables name.
  • Control flow is the order in which a computer executes code in a script.
  • IF statements will check whether a condition evaluates to be 'true' or 'false' and the answer determines the next course of action in the application.
  • An array is a single variable that is used to hold a group of data. Arrays are useful tools for storing multiple items with different data types in a single variable. Not all arrays must have multiple data types.
  • A For loop includes a starting point for the loop, the condition and what allows the array to iterate over each item.
  • A function is a set of instructions that tells the computer how to perform a certain task.
